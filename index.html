I am here to describe Semantic Elements.

A Semantic Element is a clear description of meaning for both browser and the developer. This provides a clear method of description and should help mitigate the overuse of div , which inherently tells nothing of its content. The purpose behind this is to make a page's source code more accessible by simplifying interpretation of the language.

tl/dr:

  • Elements with a meaning
  • Divitis - Avoid it!
  • HTML rocks!!!

Sites need to be able to interact in one single, universal space.
